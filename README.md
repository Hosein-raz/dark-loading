## 📖 About The Project

A sleek, wave-like dark-themed loading animation built entirely with raw HTML and CSS. By utilizing advanced CSS `@keyframes`, flexbox layouts, and staggered animation delays, this project creates a smooth, continuous pulsing effect. It's a lightweight and highly performant solution for handling loading states in modern web applications without relying on JavaScript.

## ✨ Key Technical Features

Based on the source code, this loader demonstrates several core frontend concepts:

* **Pure CSS Animation:** Zero JavaScript dependencies. The entire logic is handled via CSS `@keyframes`.
* **Staggered Delays:** Clever use of `nth-child` selectors with incrementing `animation-delay` (from `0s` to `1.2s`) to create a fluid, wave-like sequence across 7 span elements.
* **Complex Transformations:** Combines `translateX` and `scale` properties to give the elements a dynamic 3D-like movement and growth.
* **Dynamic Shadows & Opacity:** Uses `box-shadow` and `opacity` transitions seamlessly within the animation cycle to enhance the visual depth against the `#262626` dark background.
* **Flexbox Centering:** Clean and responsive centering using `display: flex; justify-content: center; align-items: center;`.

## 🚀 How to Use

Integrating this loader into your project is extremely simple:

1. Clone the repository:
  ```bash
  git clone [https://github.com/Hosein-raz/dark-loading.git](https://github.com/Hosein-raz/dark-loading.git)
  ```

2. Copy the ```<section>``` and its ```<span>``` elements from ```index.html``` into your application.

3. Import or copy the styles from style.css into your project's stylesheet.
(Note: You can easily change the #262626 background color in the CSS file to match your project's specific dark theme).

## 📂 Project Structure

```
dark-loading/
├── index.html    # Contains the 7 animated spans inside a flex container
└── style.css     # Contains the reset, layout, and keyframe animation logic
```
